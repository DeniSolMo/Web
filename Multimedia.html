Etiquetas para crear tablas




ul

La etiqueta (ul) define una lista no ordenada en HTML, donde los elementos de la lista se presentan con viñetas (puntos) por defecto. Se utiliza para agrupar una colección de elementos de manera que no haya un orden específico de prioridad o secuencia. Dentro de (ul), cada elemento de la lista se coloca dentro de una etiqueta (li). Es ideal para listas de ítems que no requieren un orden numérico, como características o tareas.

Sintaxis Ejemplo
<ul > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de UL</title>
                                        </head>
                                        <body>
                                            <ul>
                                                <li>Elemento de lista 1</li>
                                                <li>Elemento de lista 2</li>
                                                <li>Elemento de lista 3</li>
                                            </ul>
                                        </body>
                                        </html>



ol

La etiqueta (ol) se utiliza para crear una lista ordenada, donde los elementos se presentan con números o letras que indican su posición secuencial. Cada elemento de la lista se coloca dentro de una etiqueta (li) y se numera automáticamente por el navegador. Es útil cuando el orden de los elementos es importante, como en instrucciones paso a paso o listas de clasificación. También se puede personalizar el tipo de numeración con el atributo type.

Sintaxis Ejemplo
<ol> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de OL</title>
                                        </head>
                                        <body>
                                            <ol>
                                                <li>Elemento ordenado 1</li>
                                                <li>Elemento ordenado 2</li>
                                                <li>Elemento ordenado 3</li>
                                            </ol>
                                        </body>
                                        </html>



li

La etiqueta (li) define un elemento dentro de una lista, ya sea ordenada (ol) o no ordenada (ul). Cada (li) representa un ítem individual en la lista y puede contener texto, enlaces, imágenes u otros elementos HTML. Es la unidad básica que compone el contenido de las listas en HTML. Las listas anidadas se crean colocando (ul) o (ol) dentro de un (li), creando así sublistas. Es una herramienta clave para estructurar contenido en listas.

Sintaxis Ejemplo
<li > 
                                 <html>
                                 <head>
                                     <title>Ejemplo de LI</title>
                                 </head>
                                 <body>
                                     <ul>
                                         <li>Elemento de lista 1</li>
                                         <li>Elemento de lista 2</li>
                                         <li>Elemento de lista 3</li>
                                     </ul>
                                 </body>
                                 </html>



dl

La etiqueta (dl) se utiliza para definir una lista de definición, que es un tipo especial de lista en HTML. Esta lista consta de pares de términos y sus descripciones. Dentro de (dl), se agrupan términos (dt) y descripciones (dd), lo que es útil para glosarios, diccionarios, o listas de características y valores. No tiene un formato visual predeterminado, pero se utiliza comúnmente para listas donde los elementos necesitan explicación adicional.

Sintaxis Ejemplo
<dl> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DL</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



dt

La etiqueta (dt) define un término en una lista de definición (dl). Se utiliza para representar el término que será definido o descrito por el siguiente elemento (dd). En un glosario, por ejemplo, (dt) podría contener una palabra que luego se explica en el (dd) correspondiente. Es parte del par clave que forma una lista de definición, y se utiliza para identificar el elemento que se está definiendo.

Sintaxis Ejemplo
<dt> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DT</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



dd

La etiqueta (dd) se utiliza para proporcionar la descripción o definición de un término definido por (dt) en una lista de definición (dl). Aparece justo después del término al que describe y generalmente se indenta para diferenciarse visualmente. Es útil para proporcionar detalles adicionales o explicaciones sobre el término presentado en (dt). Las listas de definición son ideales para glosarios, términos y condiciones, o especificaciones de productos.

Sintaxis Ejemplo
<dd > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DD</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>

Etiquetas para crear multimedia y scripts


script

La etiqueta (script) se utiliza para incluir y ejecutar código JavaScript en una página web. Puede contener el código directamente entre las etiquetas o referenciar un archivo JavaScript externo mediante el atributo src. Es esencial para agregar interactividad, manipular el DOM, y realizar tareas como validaciones de formularios o animaciones. También se pueden especificar atributos como async y defer para controlar cómo y cuándo se carga el script. Es una herramienta fundamental para el desarrollo web dinámico.

Sintaxis Ejemplo
<script> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Script</title>
                                        <script type="text/javascript">
                                            function showAlert() {
                                                alert('¡Hola! Este es un mensaje de alerta.');
                                            }
                                        </script>
                                    </head>
                                    <body>
                                        <button onclick="showAlert()">Mostrar Alerta</button>
                                    </body>
                                    </html>

noscript

La etiqueta (noscript) se utiliza para proporcionar contenido alternativo a los usuarios cuyos navegadores no soportan JavaScript o que lo tienen deshabilitado. Lo que se coloca dentro de (noscript) solo se mostrará si el navegador no puede ejecutar scripts. Es útil para mejorar la accesibilidad y garantizar que todos los usuarios puedan acceder al contenido esencial de la página, independientemente de las capacidades de su navegador. Puede contener texto, enlaces, o cualquier otro contenido HTML básico.

Sintaxis Ejemplo
<noscript> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Noscript</title>
                                    </head>
                                    <body>
                                        <script type="text/javascript">
                                            // Este script se ejecuta si JavaScript está habilitado
                                            document.write('JavaScript está habilitado.');
                                        </script>
                                        <noscript>
                                            <p>JavaScript no está habilitado en tu navegador. 
                                            Algunas funcionalidades de esta página pueden no funcionar correctamente.</p>
                                        </noscript>
                                    </body>
                                    </html>

canvas

La etiqueta (canvas) se utiliza para dibujar gráficos y renderizar imágenes dinámicas mediante JavaScript. Es un lienzo HTML en el que se pueden dibujar líneas, formas, gráficos, y animaciones utilizando una API de gráficos basada en píxeles. No tiene contenido visible por defecto; todo lo que aparece en (canvas) debe ser generado mediante scripting. Es ideal para juegos, visualizaciones de datos, y cualquier aplicación que requiera gráficos interactivos en la web. Su contenido puede ser personalizado y manipulado de manera programática.

Sintaxis Ejemplo
<canvas> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Canvas</title>
                                        <style>
                                            canvas {
                                                border: 1px solid #000;
                                            }
                                        </style>
                                    </head>
                                    <body>
                                        <canvas id="myCanvas" width="300" height="200"> 
                                            Tu navegador no soporta el elemento canvas.
                                        </canvas>
                                        <script type="text/javascript">
                                            var canvas = document.getElementById('myCanvas');
                                            var ctx = canvas.getContext('2d');
                            
                                            // Dibuja un rectángulo rojo
                                            ctx.fillStyle = 'red';
                                            ctx.fillRect(10, 10, 150, 100);
                            
                                            // Dibuja un círculo azul
                                            ctx.beginPath();
                                            ctx.arc(200, 100, 50, 0, 2 * Math.PI);
                                            ctx.fillStyle = 'blue';
                                            ctx.fill();
                                        </script>
                                    </body>
                                    </html>

audio

La etiqueta (audio) se utiliza para incrustar archivos de audio en una página web. Puede reproducir formatos de audio como MP3, Ogg, y WAV, y admite atributos como controls para mostrar los controles de reproducción, autoplay para iniciar la reproducción automáticamente, y loop para repetir el audio. Dentro de (audio), se pueden incluir etiquetas (source) para especificar múltiples fuentes de audio, permitiendo la compatibilidad con diferentes navegadores. Es útil para agregar narraciones, música, o efectos de sonido a una página web.

Sintaxis Ejemplo
<audio> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Audio</title>
                                    </head>
                                    <body>
                                        <audio controls>
                                            <source src="audio-file.mp3" type="audio/mpeg">
                                            Tu navegador no soporta el elemento audio.
                                        </audio>
                                    </body>
                                    </html>

video

La etiqueta (video) se utiliza para incrustar archivos de video en una página web. Admite varios formatos de video como MP4, WebM, y Ogg, y permite la reproducción mediante controles incorporados, que incluyen play, pause, y volumen. Al igual que (audio), soporta atributos como controls, autoplay, y loop. Dentro de (video), se pueden incluir varias etiquetas (source) para proporcionar diferentes versiones del video, optimizando la compatibilidad entre navegadores. Es clave para integrar contenido multimedia de manera eficiente.

Sintaxis Ejemplo
<video> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Video</title>
                                    </head>
                                    <body>
                                        <video width="640" height="360" controls>
                                            <source src="video-file.mp4" type="video/mp4">
                                            <source src="video-file.webm" type="video/webm">
                                            Tu navegador no soporta el elemento video.
                                        </video>
                                    </body>
                                    </html>

source

La etiqueta (source) se utiliza dentro de (audio), (video), o (picture) para especificar diferentes archivos de medios que el navegador puede utilizar. Permite definir múltiples fuentes para un solo archivo multimedia, lo que garantiza que el navegador use la mejor opción compatible. A través del atributo src, se indica la ubicación del archivo, y mediante type, se especifica el tipo de medio, como audio/mpeg o video/mp4. Es esencial para asegurar la correcta reproducción de contenido multimedia en diferentes navegadores y dispositivos.

Sintaxis Ejemplo
<source> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Source</title>
                                        </head>
                                        <body>
                                            <video width="640" height="360" controls>
                                                <source src="video-file.mp4" type="video/mp4">
                                                <source src="video-file.webm" type="video/webm">
                                                Tu navegador no soporta el elemento video.
                                            </video>
                                        </body>
                                        </html>

track

La etiqueta (track) se utiliza para especificar pistas de texto temporizadas en medios como video o audio. Estas pistas pueden incluir subtítulos, descripciones, capítulos, o metadatos. Se coloca dentro de una etiqueta (video) o (audio) y se asocia al archivo multimedia mediante el atributo src, que define la ubicación del archivo de pista. Además, se pueden usar atributos como kind para especificar el tipo de pista, label para nombrarla, y srclang para indicar el idioma. Es esencial para mejorar la accesibilidad y permitir a los usuarios seleccionar subtítulos en diferentes idiomas.

Sintaxis Ejemplo
<track> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Track</title>
                                    </head>
                                    <body>
                                        <video width="640" height="360" controls>
                                            <source src="video-file.mp4" type="video/mp4">
                                            <track kind="subtitles" src="subtitles-en.vtt" srclang="en" label="English">
                                            <track kind="descriptions" src="descriptions-en.vtt" srclang="en" label="English">
                                            <track kind="chapters" src="chapters-en.vtt" srclang="en" label="English">
                                            Tu navegador no soporta el elemento video.
                                        </video>
                                    </body>
                                    </html>

iframe

La etiqueta (iframe) se utiliza para incrustar otra página web dentro de la página actual. Funciona como un contenedor que muestra un documento HTML independiente, el cual puede provenir del mismo sitio o de un dominio externo. A través de atributos como src, se define la URL del contenido a incrustar, y con width y height, se ajustan las dimensiones del marco. Es útil para integrar contenido externo como videos de YouTube, mapas interactivos, o aplicaciones web. Los iframes pueden afectar el rendimiento y la seguridad, por lo que deben usarse con cuidado.

Sintaxis Ejemplo
<iframe> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Iframe</title>
                                    </head>
                                    <body>
                                        <iframe src="https://www.example.com" width="600" height="400" frameborder="0">
                                            Este contenido no puede ser mostrado porque tu navegador no soporta iframes.
                                        </iframe>
                                    </body>
                                    </html>

embed

La etiqueta (embed) se utiliza para incrustar contenido multimedia o aplicaciones en la página web. Es similar a (iframe) pero está más enfocada en incluir objetos que no son HTML, como archivos Flash, videos, o incluso contenido de plugins. El atributo src especifica la URL del recurso a incrustar, mientras que width y height definen su tamaño. Es una etiqueta auto-cerrada y no tiene contenido interno. Es una forma rápida de integrar contenido externo, pero como (iframe), ebe usarse con precaución para evitar problemas de seguridad.

Sintaxis Ejemplo
<embed> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Embed</title>
                                    </head>
                                    <body>
                                        <embed src="example-file.pdf" type="application/pdf" width="600" height="400" />
                                        <embed src="example-video.mp4" type="video/mp4" width="600" height="400" />
                                    </body>
                                    </html>

object

La etiqueta (object) es un contenedor versátil para incrustar cualquier tipo de contenido multimedia, como imágenes, videos, aplicaciones, o documentos PDF. Permite especificar un recurso externo mediante el atributo data, que indica la URL del archivo a mostrar, y type, que define el tipo MIME del recurso. Dentro de (object), se pueden incluir varias etiquetas (param) para proporcionar parámetros adicionales al objeto incrustado. Es más flexible que (embed) y (iframe), permitiendo una mayor integración de contenido interactivo y multimedia.

Sintaxis Ejemplo
<object> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Object</title>
                                    </head>
                                    <body>
                                        <object data="example-file.pdf" type="application/pdf" width="600" height="400">
                                            
Tu navegador no soporta la visualización de PDFs. Puedes descargar el archivo aquí.

                                        </object>
                                        
                                        <object data="example-video.mp4" type="video/mp4" width="600" height="400">
                                            
Tu navegador no soporta la visualización de este video. Puedes descargar el archivo aquí.

                                        </object>
                                    </body>
                                    </html>

param

La etiqueta (param) se utiliza dentro de (object) para pasar parámetros específicos al recurso incrustado. Cada (param) define un par clave-valor, donde name especifica el nombre del parámetro y value define su valor. Es esencial para controlar cómo se presenta o se comporta el contenido incrustado, permitiendo ajustes como la configuración de un video, la definición de un ID de documento, o la personalización de una aplicación Flash. Aunque es una etiqueta auto-cerrada, es crucial para proporcionar opciones adicionales al objeto.

Sintaxis Ejemplo
<param> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Param</title>
                                    </head>
                                    <body>
                                        <object data="example-file.swf" type="application/x-shockwave-flash" width="600" height="400">
                                            <param name="movie" value="example-file.swf">
                                            <param name="quality" value="high">
                                            <param name="bgcolor" value="#ffffff">
                                            
Tu navegador no soporta el objeto Flash. Puedes descargar el archivo aquí.

                                        </object>
                                    </body>
                                    </html>

map

La etiqueta (map) se utiliza para definir un mapa de imagen, que es una imagen con áreas seleccionables que actúan como enlaces. Dentro de (map), se definen varias etiquetas (area) que especifican las regiones interactivas de la imagen, asociadas a enlaces mediante el atributo href. Cada (area) se puede definir en forma de rectángulos, círculos, o polígonos para delinear partes específicas de la imagen. El atributo name de (map) se enlaza con el atributo usemap de una imagen para asociar la imagen con el mapa interactivo. Es útil para crear gráficos navegables o imágenes con enlaces específicos.

Sintaxis Ejemplo
<map> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Map</title>
                                    </head>
                                    <body>
                                        <img src="example-image.jpg" usemap="#image-map" width="600" height="400" alt="Imagen con mapa de imagen">
                                        
                                        <map name="image-map">
                                            <area shape="rect" coords="34,44,270,350" href="https://www.example.com" alt="Enlace a Example.com">
                                            <area shape="circle" coords="200,200,50" href="https://www.example.org" alt="Enlace a Example.org">
                                            <area shape="poly" coords="300,100,400,150,350,200" href="https://www.example.net" alt="Enlace a Example.net">
                                        </map>
                                    </body>
                                    </html>

area

La etiqueta area se utiliza dentro de un elemento map para definir áreas activas en una imagen de mapa de imagen. Estas áreas son zonas clicables que pueden redirigir a diferentes URLs o activar otras acciones. Cada area define una región de la imagen y su comportamiento asociado, como enlaces o scripts.

Sintaxis Ejemplo
<area> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Area</title>
                                    </head>
                                    <body>
                                        <img src="example-image.jpg" usemap="#image-map" width="600" height="400" alt="Imagen con mapa de imagen">
                                        
                                        <map name="image-map">
                                            <area shape="rect" coords="34,44,270,350" href="https://www.example.com" alt="Enlace a Example.com">
                                            <area shape="circle" coords="200,200,50" href="https://www.example.org" alt="Enlace a Example.org">
                                            <area shape="poly" coords="300,100,400,150,350,200" href="https://www.example.net" alt="Enlace a Example.net">
                                        </map>
                                    </body>
                                    </html>

progress

La etiqueta progress se utiliza para mostrar el progreso de una tarea o proceso en curso. Es un elemento visual que presenta una barra de progreso que indica cuánto se ha completado de una tarea. Se usa comúnmente para representar la carga de archivos, el progreso de una descarga o el avance en una operación de procesamiento.

Sintaxis Ejemplo
<progress> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Progress</title>
                                    </head>
                                    <body>
                                        <h2>Barra de Progreso</h2>
                                        
                                        <progress value="70" max="100" style="width: 300px;">70%</progress>
                                        
                                        <p>Esta barra de progreso indica un progreso del 70%.</p>
                                    </body>
                                    </html>

meter

La etiqueta meter se emplea para representar una medida escalar dentro de un rango conocido. A diferencia de progress, que se enfoca en el progreso, meter se usa para mostrar valores de medición como la cantidad de batería restante, la temperatura o el nivel de un recurso.

Sintaxis Ejemplo
<meter> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Meter</title>
                                    </head>
                                    <body>
                                        <h2>Medidor</h2>
                                        
                                        <meter value="0.6" min="0" max="1" low="0.2" high="0.8" optimum="0.5" style="width: 300px;">60%</meter>
                                        
                                        <p>Este medidor muestra un valor de 60% con una gama de valores bajos, altos y óptimos.</p>
                                    </body>
                                    </html>

code

La etiqueta code se emplea para denotar un fragmento de código de computadora dentro de un texto. Esta etiqueta se usa principalmente en documentos técnicos o tutoriales para destacar el código fuente, ya sea en HTML, CSS, JavaScript u otros lenguajes de programación.

Sintaxis Ejemplo
<code> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Code</title>
                                    </head>
                                    <body>
                                        <h2>Código en Línea</h2>
                                        
                                        <p>Aquí hay un ejemplo de código en línea usando la etiqueta <code>.</p>
                                        <p>El código de ejemplo es: <code>console.log('Hola, mundo!');</code></p>
                                        
                                        <h2>Código en Bloque</h2>
                                        <pre>
                                            <code>
                                            function greet() {
                                                console.log('Hola, mundo!');
                                            }
                                            greet();
                                            </code>
                                        </pre>
                                    </body>
                                    </html>

kbd

La etiqueta kbd se utiliza para representar la entrada del teclado del usuario en un documento HTML. Es útil para mostrar qué combinaciones de teclas deben presionarse para realizar una acción o comando específico. El texto dentro de kbd se presenta típicamente en una fuente monoespaciada para emular el aspecto de las teclas del teclado.

Sintaxis Ejemplo
<kbd> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de KBD</title>
                                    </head>
                                    <body>
                                        <h2>Entrada del Teclado</h2>
                                        
                                        <p>Para copiar y pegar texto, presiona <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar y <kbd>Ctrl</kbd> + <kbd>V</kbd> para pegar.</p>
                                        
                                        <p>Para guardar un archivo, utiliza <kbd>Ctrl</kbd> + <kbd>S</kbd>.</p>
                                    </body>
                                    </html>

samp

La etiqueta samp se usa para representar una muestra de salida de un programa o una aplicación. Es comúnmente utilizada en documentación técnica para mostrar el resultado o la salida de un código de ejemplo. El contenido dentro de samp generalmente se muestra en una fuente monoespaciada, similar a la usada para el código, para mantener la claridad en la presentación de los datos de salida.

Sintaxis Ejemplo
<samp> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Samp</title>
                                    </head>
                                    <body>
                                        <h2>Salida de Ejemplo</h2>

                                        <p>La salida del programa es: <samp>Error: archivo no encontrado</samp>.</p>
                                        
                                        <p>Otra salida del programa es: <samp>Conexión exitosa</samp>.</p>
                                    </body>
                                    </html>

var

La etiqueta var se usa para denotar una variable en un contexto matemático o de programación dentro de un documento HTML. Permite resaltar el texto que representa una variable en fórmulas o fragmentos de código, ayudando a la comprensión del contenido técnico. El texto dentro de var se muestra en cursiva por defecto, lo que ayuda a distinguirlo visualmente del resto del texto

Sintaxis Ejemplo
<var> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Var</title>
                                    </head>
                                    <body>
                                        <h2>Variable</h2>
